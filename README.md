# StadiumGenie AI

StadiumGenie AI is a Generative AI-powered assistant designed for FIFA World Cup 2026 stadium operations.

## Features

- 🌍 Multilingual AI Assistant
- 🗺️ Smart Stadium Navigation
- 👥 Crowd Management Support
- 🚨 Emergency Assistance
- 🚌 Transport Guidance
- ♿ Accessibility Support
- 📊 Operations Intelligence

## Tech Stack

- Python (Flask)
- Google Gemini API
- Google Cloud Run
- Docker

## Run Locally

### Install dependencies

```bash
pip install -r requirements.txt
```

### Set your Gemini API Key

Replace `YOUR_GEMINI_API_KEY` in `app.py`.

### Run

```bash
python app.py
```

Open:

```
http://localhost:5000
```

## Deploy

Deploy using Google Cloud Run.
